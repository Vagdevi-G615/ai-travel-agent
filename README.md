An intelligent AI Travel Agent that answers complex travel-related queries using Retrieval-Augmented Generation (RAG). The system retrieves relevant travel knowledge from a vector database and generates accurate, context-aware responses using a Large Language Model.

### Key Features  

- Semantic search using FAISS
- Context-aware answers via RAG
- Fast inference using Groq-hosted LLaMA models
- Interactive UI built with Streamlit
- Deployed on Hugging Face Spaces
- Secure API key handling via environment secrets

### Tech Stack

- Python
- Retrieval-Augmented Generation (RAG)
- FAISS Vector Database
- Sentence Transformers
- Groq LLM (LLaMA)
- Streamlit
- Hugging Face Spaces
- Docker
